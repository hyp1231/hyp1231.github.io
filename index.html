题意

$n \le 1e5, m \le 2e6$

输入

第一行两个正整数 $n$ 和 $m$。

记上一个回答为 $ans$，实际的边为 $(u, v)$
则有 $\begin{cases}u’ = (u + ans) \pmod n + 1\\ v’ = (v + ans) \pmod n + 1\end{cases}$
